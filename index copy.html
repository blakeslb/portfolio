Projects

Web Applications

The Natural Hair Directory

The Natural Hair Directory is a dynamic web app that provides a map-based directory of hair stylists and salons who specialize in Black and natural haircare. Built with RESTFUL routes, CRUD operations, NodeJS and a noSQL (MongoDB) database. The directory uses the Google Maps Platform API and geocoding to automatically populate makers on the map based on user submission. The application is responsive to all screen sizes (mobile, tablet, desktop) and was developed with a mobile-first design.

Codesquad Comics

Codesquad Comics is a web application built to the specification of the Codesquad final project. This project was designed to be fully responsive. Features include OAuth authentication, user registration and login in, and the ability to create, update delete comics. Each comic card links to a separate details page that uses EJS templates to pass information from a MongoDB database.

Internal Tool

In 2021, during the time I was transitining from social work to web development, I worked for a great therapy referral service in Massachusetts. Users would submit forms to a site requesting therapy referrals based on a set of criteria.Even during the pandemic, location was usually a primary concern. Staff often needed t query the database to look for therapists by regional codes of the preferred location as well as surrounding locations. To make this search easier, specially for those of us with rusty geography, I created an internal tool to display a map of MA cities and their corresponding regional codes in hover states through simple vanilla JS and CSS
